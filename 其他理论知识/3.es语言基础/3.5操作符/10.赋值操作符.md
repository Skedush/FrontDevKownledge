### 赋值操作符

​	简单赋值用等于号(=)表示，将右手边的值赋给左手边的变量，如下所示:

```javascript
let num = 10;
```

​	复合赋值使用乘性、加性或位操作符后跟等于号(=)表示。这些赋值操作符是类似如下常见赋值操作的简写形式:

```javascript
let num = 10;
num = num + 10;
```

​	以上代码的第二行可以通过复合赋值来完成:

```javascript
let num = 10;
num += 10;
```

​	每个数学操作符以及其他一些操作符都有对应的复合赋值操作符:

	- 乘后赋值(*=)
	-  除后赋值(/=)
	- 取模后赋值(%=)
	- 加后赋值(+=)
	- 减后赋值(-=)
	- 左移后赋值(<<=)
	- 右移后赋值(>>=)
	- 无符号右移后赋值(>>>=)

​	这些操作符仅仅是简写语法，使用它们不会提升性能。
